Big Data Semantic Management for Disease Prediction

Installation
------------
To install the required packages for this project, navigate to the project directory and run the following command in your terminal:

pip install -r requirements.txt

Running the Project
-------------------
To run this project, you need to execute the `app.py` file. Follow these steps:

1. Ensure all dependencies are installed by following the installation instructions above.
2. Navigate to the project directory in your terminal.
3. Run the application using Python:
   python app.py
4. Access the application through your web browser at `http://localhost:5000`.

Abstract
--------
In the rapidly evolving landscape of healthcare, the fusion of machine learning and data science has ushered in a new era, revolutionizing the possibilities for accurate and timely medical diagnostics. This paradigm shift has prompted the exploration of innovative frameworks, with a particular focus on comprehensive solutions that integrate cutting-edge technologies to address the complexities of diseases such as Diabetes, Breast Cancer, Alzheimer's, Skin Cancer lesions, and Malaria. Our proposed model seamlessly integrates image recognition and a user-friendly menu-based system, utilizing logistic regression for Diabetes and Breast Cancer prediction, Convolutional Neural Networks (CNNs) for Alzheimer's and skin cancer analysis, and contour detection techniques for Malaria classification. Through a rigorous evaluation of diverse medical images and test reports, our model demonstrates a commendable level of accuracy and efficiency in the diagnosis and classification of these diseases, underscoring its potential as a dependable tool for healthcare professionals. The transformative impact of this integrated model on the landscape of medical diagnostics emphasizes the need for continued research and development, further highlighting the potential for advancements in healthcare technology and patient-centric care.
